# Django migration branch (инструкция)

Этот архив содержит базовую структуру Django-приложения `hotel` и вспомогательные файлы,
чтобы вы могли быстро создать ветку `django-migration` и интегрировать изменения.

Инструкции по использованию см. внутри (создайте проект Django, поместите папку hotel в корневой каталог репозитория и следуйте шагам).
